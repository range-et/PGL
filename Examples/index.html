Plebeian Graph Library

3D graph visualization with Three.js — examples

Example 1
Simple ZKC

Basic 3D graph: ZKC dataset with box nodes and thick edges. Good starting point for loading data and drawing a graph.
Example 2
Edge bundling

ZKC with curved, bundled edges and vertical displacement for an arc-style flow look. Combines DrawEdgeBundling and DisplaceEdgeInY.
Example 3
Large point cloud

Erdős–Rényi random graph (5000 nodes), Kamada–Kawai layout, and point-cloud + thin edges. Scales to larger graphs.
Example 4
Toggle activation

Interactive node highlighting: click to toggle a subset “active” and recolor nodes. Useful for diffusion or selection demos.
Example 5
Hierarchy

3D KD-tree clustering: nodes within a distance threshold are merged into super-nodes at centroids. Simplified graph with thick edges.
Example 6
Flow map

FlowmapBlue-style: zoom in/out to change clustering. Zoom out = fewer merged nodes and flows; zoom in = more detail. 3D KD-tree by camera distance.
Example 7
Graph algorithms

BFS, Dijkstra, SelectSubgraph: path from a start node highlighted in green/red, and a subgraph drawn. Shows GraphMethods usage.
Example 8
ThreeWrapper variants

Different draw styles: box nodes, simplified (cylinder-style) edges, and modularity-based node/edge groups by a node property.
